3. Definitions

A
Rows
A.1
Row

A sequence of bells in which each bell rings once and only once.

Example: 531246 is a 6-bell Row.

Further explanation: Bells are incrementally numbered in order from the bell with the highest pitch to the bell with the lowest pitch, and may be referred to using either cardinal or ordinal numbers. Number 1 is used for the bell with the highest pitch, number 2 for the next highest pitch and so on.

When using ordinal numbers, bell number 2 becomes 'the 2nd', bell number 3 becomes 'the 3rd', and so on. However bell number 1 is usually referred to as 'the treble' rather than 'the 1st', and the bell with the lowest pitch is usually referred to as 'the tenor' rather than the corresponding ordinal.

The numbering of bells always relates solely to the set of bells comprising the Row, ignoring any other available bells that are not being used at the time.

Rows are normally shown in columns so that the path of individual bells can be visualised by looking down the columns. In order to keep the columns aligned, a fixed font can be used with single characters representing each bell. Conventionally '0' (zero) is used to represent the 10th, and upper case letters (excluding 'I', 'O' and 'X') are used for bells 11 to 33 as follows:

'E' (Eleven); 'T' (Twelve); 'A' (Thirteen); 'B' (Fourteen); 'C' (Fifteen); 'D' (Sixteen); 'F' (Seventeen); 'G' (Eighteen); 'H' (Nineteen); 'J' (Twenty); 'K' (Twenty-one); 'L' (Twenty-two); 'M' (Twenty-three); 'N' (Twenty-four); 'P' (Twenty-five); 'Q' (Twenty-six); 'R' (Twenty-seven); 'S' (Twenty-eight); 'U' (Twenty-nine); 'V' (Thirty); 'W' (Thirty-one); 'Y' (Thirty-two); 'Z' (Thirty-three)

Technical comments: This definition of Row precludes cylindrical ringing, wherein a bell can ring twice in some rows and not at all in other rows. Cylindrical ringing is considered beyond the scope of this framework -- it would likely require its own framework, with modified definitions in areas such as truth.

A.2
Rounds

A Row in which the bells ring in the order from highest pitch to lowest pitch.

Example: 12345678 is Rounds for an 8-bell Row.

A.3
Place

The position of a bell within a Row.

Example: In the Row 142536, the 2nd is ringing in 3rd's Place.

Further explanation: Places are numbered starting with 1 for the bell that rings first, and increasing sequentially for each next bell to ring. E.g. the bell that rings second in a Row is in Place 2.

Each bell in a Row rings in a different Place to all other bells in that Row, and therefore there are the same number of Places in a Row as there are bells in that Row.

Places are usually referred to using ordinal numbers in the possessive form, so Place 2 is usually '2nd's Place'. 2nd's Place is a reference to the Place the 2nd bell occupies when ringing Rounds.

The bell in 1st's Place is often said to be Leading, and the bell in the last Place of the Row is often said to be Lying. E.g. in the Row 214365, the 2nd is Leading and the 5th is Lying.

Note that in a Row containing one or more Cover Bells (see Section D), the numbering of the Places may be modified to exclude the Cover Bell(s). E.g. when ringing Triples (see Section H) with a Cover Bell in 8th's Place, the bell in 7th's place may be said to be Lying.

Technical comments: More formally, a Place is a position where an individual bell can ring within a Row, referenced in the order of the first bell to ring to the last bell to ring, with one Place in a Row for every bell in that Row.

B
Changes
B.1
Change

A transposition of bells from one Row to the next Row, where both Rows are made up of the same set of bells.

Example: A Change might transpose the bells from the Row 214365 to the Row 241356.

Further explanation: A Change transposes a bell to a unique Place in the next Row -- i.e. a Change does not transpose two or more bells from one Row into the same Place in the next Row.

Changes are often represented using Place Notation. This notation is formed by writing the Places of bells that remain in the same Place from one Row to the next Row, working from beginning to end in the Row. All remaining bells swap Places with the bell next to them.

E.g. for 4-bell Changes, the place notation ‘14’ indicates that the Leading and Lying bells stay in the same Place across Rows, and the bells in 2nd's and 3rd's Places swap. So if Change '14' is applied to Row 2143, the resulting Row would be 2413.

In Place Notation, single character representations of bells are used for Places above 9th's Place (e.g. "9T"). The symbols 'x' or '-' are used to denote a Change on an even number of bells where no bell stays in the same Place, so all pairs of bells swap.

Dots are used to delineate between consecutive Changes where at least one bell remains in the same Place in both Changes (e.g. '14.58'). '14.58' (when referring, e.g., to 8-bell Changes) can optionally be represented as '4.5' since the '1' and '8' can be implied.

B.1.1
Identity Change

A Change that transposes each bell to the same Place in a Row as it occupied in the previous Row.

Example: The Change that transposes the bells from Row 54321 to Row 54321 is an Identity Change.

B.1.2
Adjacent Change

A Change that transposes each bell either to the same Place in a Row as it occupied in the previous Row, or to a Place in a Row that is immediately adjacent to the Place it occupied in the previous Row, but which is not an Identity Change.

Example: The Change that transposes the bells from Row 21436587 to Row 12346857 is an Adjacent Change.

B.1.3
Jump Change

A Change that is neither an Identity Change nor an Adjacent Change.

Example: The Change that transposes the bells from Row 12345678 to Row 23415678 is a Jump Change because the Treble moves from 1st's Place to 4th's Place, which are not adjacent Places.

Further explanation: Note that for 1-bell Rows, the only possible Change is an Identity Change. For 2-bell Rows, Identity and Adjacent Changes are possible. For Rows with 3 or more bells, all three types of Changes are possible.

Also note that Place Notation can only be used to describe an Adjacent Change or an Identity Change, or a sequence of such Changes. Place Notation does not support Jump Changes.

B.2
Change Ringing

The practice of ringing Rows where the progression between Rows is defined by Changes.

Further explanation: Call Changes (or 'Called Changes') is a form of Change Ringing, but not a form of Method Ringing.

C
Methods
C.1
Method

A sequence of Changes, all operating on the same set of bells, that is used to produce a sequence of resulting Rows from an initial Row.

Example: x16x14x16x12 (Little Bob Minor).

Further explanation: Note that if the last Change of Little Bob Minor (the '12') is removed, the remaining sequence of Changes (i.e. x16x14x16x) is palindromic, meaning it is the same if read forwards or backwards. Where a Method has palindromic symmetry, it may be recorded in shorthand in a Method Library in a form such as &x16x14+12. In this example, the '&' indicates that the portion before the '+' must also be reversed (except for the last change) to produce the palindromic section, before then adding the portion after the '+'.

C.1.1
Static Method

A Method whose sequence of Changes is fixed and finite.

Example: x16x14x16x12 is an example of a Static Method on 6 bells. The Method can be represented by a fixed length place notation.

C.1.2
Dynamic Method

A Method that is not a Static Method.

Example: Dixon's Bob Minor.

Further explanation: Dixon's Bob Minor specifies that Rows are generated by successively applying the pair of Changes x16 except (a) if the Treble is leading after an x Change, replace the next 16 Change with a 12 Change, and (b) if the 2nd or 4th is leading after an x Change, replace the next 16 Change with a 14 Change.

In this Method, the sequence of Changes varies depending on the Rows generated, so using a different starting Row could generate a different sequence of Changes. This Method's sequence of Changes is therefore not fixed, and so Dixon's Bob Minor is an example of a Dynamic Method.

C.2
Method Name

A name given to a Method that is unique for its Class and Stage.

Example: x16x14x16x12 (Little Bob Minor).

Further explanation: A Method can be named when the requirements of Section N have been met.

C.3
Method Title

The full description of a Method formed by concatenating its Method Name, Class Name and Stage Name.

Example: Plain Bob Doubles. 'Plain' is the Method Name, 'Bob' is the Class, and 'Doubles' is the Stage Name.

C.4
CCCBR Method Library

A central collection, maintained by the CCCBR, of all named Methods recognized by the CCCBR.

C.5
Rotation

A Rotation is a Method whose Changes are the same as another Method, except that the Changes, when considered as a cycle, have been rotated to start at a different point in the cycle.

Example: The Method Plain Bob Minimus has the Changes -14-14-14-12. The Method with the Changes -14-12-14-14 is a Rotation of Plain Bob Minimus.

C.6
Method Ringing

The practice of ringing Rows where the progression between Rows is defined primarily by Methods, secondarily by Calls, and with Rows optionally also including one or more Cover Bells.

D
Cover Bells
D.1
Cover Bell

A bell that occupies a Place in a Row that is not one of the Places operated on by the Changes of the Method being rung.

Example: A Doubles Method is often rung with 6-bell Rows, where the Method's Changes operate on Places 1 to 5, and 6th's Place contains a Cover Bell (normally the tenor).

Technical comments: More formally, a Cover Bell:
1. Occupies a Place in a Row that is not one of the Places operated on by the Changes of the Method(s) being rung; and
2. Remains in the same Place from one Row to the next unless affected by a Call (see Section E), or affected by a change to a Method whose Changes operate on the Cover Bell’s Place.

E
Calls
E.1
Call

An instruction to replace Change(s) from a Method with different Change(s).

Example: The Change '12' when the Treble is Leading in Plain Bob Minor is replaced with the Change '14' when a 'bob' is called. A 'bob' is a commonly-used Call in Method Ringing.

Further explanation: Most commonly a call replaces a single Change with another, but a call may also add Changes, remove Changes, or replace a number of Changes with a different number of Changes.

When a Call adds or removes Changes, these Changes are respectively added to or subtracted from the related Method's total number of Changes rung for Performance Reporting purposes.

Technical comments: More formally, a Call is an instruction to replace zero or more Changes from one or more Methods with a different set of zero or more Changes. Calls may also operate on Cover Bell(s) to produce variable cover ringing.

When a Call affects the Changes of more than one Method (i.e. in Spliced), the Composition should make clear how the Change count of each Method is affected so that the Performance can be accurately reported.

F
Compositions
F.1
Composition

An arrangement of Method(s) and Call(s), all operating on the same set of bells, that is used to produce a sequence of resulting Rows from an initial Row.

Technical comments: More formally, a Composition is an arrangement of Method(s), Call(s) and/or Cover Bell(s), all operating on the same set of bells, that is used to produce a sequence of resulting Rows from an initial Row.

It is possible to use more than one Method to produce a single Row. E.g. a Composition could produce 12-bell Rows by having a Cover Bell in Place 1, applying a Doubles Method to Places 2 to 6, and applying a Minor Method to Places 7 to 12. Later Rows in the same Composition could be produced by applying a Minimus Method to Places 1 to 4, having a Cover Bell in 5th's Place, and applying a Triples Method to Places 6 to 12. A Composition specifies how the various Methods, Calls and Cover Bells it uses are applied throughout the Composition to generate Rows.

There is no requirement to use whole multiples of a Method's Changes in a Composition. Any portion of a Method's Changes can be used as part of a Composition.

G
Blocks
G.1
Block

A sequence of Rows, all composed of the same set of bells, that is produced from an initial Row using a Method or a Composition.

G.2
Round Block

A Block whose final Row is the same as its initial Row.

Further explanation: While Round Blocks are normally rung starting and ending with Rounds, they can be rung starting and ending with any Row.

G.3
Plain Lead

A Block produced, starting from an initial Row, by applying a Method's sequence of Changes once, without using any Calls.

Further explanation: The sequence of Changes for the Method 'Little Bob Minor' is x16x14x16x12. When these Changes are applied once to an initial Row, the resulting Block is one Plain Lead of that Method.

G.4
Plain Course

A Block produced, starting from an initial Row, by applying a Method's sequence of Changes repeatedly, without using any Calls, until a Round Block is obtained.


Example: This is a Plain Course of Bastow Little Bob Minimus.

Further explanation: The sequence of Changes for Bastow Little Bob Minimus is x12x14. These Changes produce one Plain Lead, bringing up the row 1423 when starting from Rounds, as shown in the diagram. If these Changes are applied twice more, Rounds is produced. Bastow Little Bob Minimus therefore has 3 Plain Leads in its Plain Course.

G.5
Length

The Length of a Block is the number of Changes that were used to produce the Block.


Further explanation: In many Blocks, both the initial Row and the final Row are Rounds. Length is measured in number of Changes instead of number of Rows to avoid the question of whether the initial Row or the final Row, or neither or both, are included when determining the Length of a Block.

In this Block, it can be seen there are a total of 9 Rows, including both the initial Rounds and the final Rounds. However, 8 Changes (as represented by Place Notations on the left) are used to produce it. This Block therefore has a Length of 8 Changes.

G.6
Fixed Cover Place

A Place in a Block in which the same Cover Bell rings in all Rows of that Block.

Further explanation: Because a Block comprises Rows that are all composed of the same set of bells, a Place in a Block refers to that same Place in all of the Block's underlying Rows.

Example: Assume Plain Bob Doubles is being rung on the front 5 of a ring of 8, and Cover Bells are ringing in 6th's, 7th's and 8th's Places. Calls are used in the Composition such that (a) occasionally the bells in 5th's and 6th's Places are called to swap, and (b) occasionally, at other times to (a), the bells in 6th's and 7th's Places are called to swap.

In this Block, the bell ringing in 8th's Place is always the same bell because it is not ringing a Method and it is never affected by a Call. 8th's Place is therefore a Fixed Cover Place.

Technical comments: While a Fixed Cover Place is defined above in terms of the same Cover Bell ringing in all Rows of a Block, the same effect can be achieved in different ways. For example, a Method may keep a bell in the same Place in every Row; or a Method may keep a bell in the same Place in most Rows, but Calls are used that keep that bell in the same Place in every Row. Regardless of how the Rows are generated, if the same bell rings in the same Place in every Row of a Block, that Place is a Fixed Cover Place.

G.7
Variable Cover Place

A Place in a Block in which a Cover Bell rings for at least one Row of that Block, but which is not a Fixed Cover Place.

Example: In the example for G.6 above, 6th's and 7th's Places are Variable Cover Places because they contain Cover Bells for at least one of the Rows (and in fact for this example, for all of the Rows), but they are not Fixed Cover Places because the same Cover Bell does not ring in either 6th's Place or 7th's Place all of the time.

Places 1 - 5 in the example for G.6 above never contain a Cover Bell, so they are neither Fixed Cover Places nor Variable Cover Places.

G.8
Extent

The complete set of distinct Rows possible for a given set of bells.

Futher explanation: The number of Rows in an Extent for a given set of bells is the factorial of the number of bells in the set. E.g. for 4 bells, the Extent has 4! Rows = 1 * 2 * 3 * 4 = 24 Rows.

Note that while a Block may contain one or more Extents, this definition of Extent neither implies that the Rows are in any particular order, nor that the Rows making up an Extent are necessarily contiguous within a Block.

H
Stages
H.1
Stage

The number of bells.

Further explanation: A term that applies to Rows, Changes, Methods, Compositions and Blocks:

The Stage of a Row is the number of bells in that Row.

The Stage of a Change is the number of bells on which that Change operates.

Since a Method comprises Changes that all operate on the same set of bells, a Method has the Stage of its underlying Changes.

Since a Composition is used to produce Rows that comprise the same set of bells, a Composition has the Stage of the Rows it produces.

Since a Block comprises Rows that are all composed of the same set of bells, a Block has the Stage of its underlying Rows.

H.2
Stage Name

Names that are given to Stages:

1 = One; 3 = Singles; 5 = Doubles; 7 = Triples; 9 = Caters; 11 = Cinques; 13 = Sextuples; 15 = Septuples; etc.

2 = Two; 4 = Minimus; 6 = Minor; 8 = Major; 10 = Royal; 12 = Maximus; 14 = Fourteen; 16 = Sixteen; etc.

H.3
Effective Stage

The number of Places remaining in a Block after any Fixed Cover Places have been excluded.

Example: In the example for G.6 above, the Effective Stage would be 7 because only 8th's Place is a Fixed Cover Place.

Further explanation: The Effective Stage of a Composition is the same as the Effective Stage of the Block it produces.

I
Truth
I.1
True

A Block is True if each of the possible Rows for the Block's Effective Stage occurs either n or n+1 times, for a given value of n greater than or equal to zero.

Further explanation: In a Round Block, either the initial Row or the final Row (but not both) is excluded when determining whether the Block is True.

In a non-Round Block, both the initial Row and the final Row are included when determining whether the Block is True.

When the number of Rows in a Block is less than or equal to the number of Rows in the Extent of the Block's Effective Stage, the definition of True simplifies to 'a Block that has no repeated Rows'. In this case, all possible Rows appear either 0 times or 1 time, so n in the definition has a value of 0.

Examples:

The first example shows a Round Block at the Singles Stage. Since there are no Fixed Cover Bells, the Effective Stage of the Block is also Singles.

There are 6 possible Rows at the Singles Stage. 4 of the possible Rows appear once in the Block, and 2 of the possible Rows appear twice in the Block.

Therefore all possible Rows appear either 1 time or 2 times, so the Block is True, and n in the definition has a value of 1.

The second example shows a non-Round Block at the Minimus Stage. Since there is a Fixed Cover Bell in 4th's Place, the Effective Stage of the Block is Singles.

There are 6 possible Rows at the Singles Stage. 3 of the possible Rows appear once in the Block, 2 of the possible Rows appear twice in the Block, and 1 of the possible Rows appears three times in the Block. (Both the initial and final Rows are considered because this is a non-Round Block.)

Since some possible Rows appear n times, others appear n+1 times, and others appear n+2 times, where n=1, the 'either n or n+1 times' requirement is not met (since there is also n+2) and so the Block is False.

J
Performances
J.1
Performance

The successful ringing of a Block where the band strives to maintain a high standard of ringing, and errors in ringing or calling are corrected quickly.

Further explanation: More fully, this term would be 'Method Ringing Performance'. Performances are often distinguished by the intent for the ringing to be heard by members of the public, and/or for the ringing to be reported to the ringing community when it is 1250 Changes or greater in Length.

'Method Ringing Practice' (or practice ringing), by contrast, is usually for the purpose of teaching and developing ringers and bands.

While Performances may be of any Length, practice ringing is usually less than 1250 Changes in Length.

J.2
Touch

A Performance or Practice of any Length.

J.3
Short Touch

A Performance or Practice whose Length is 1 to 1249 Changes.

J.4
Quarter Peal

A Performance whose Length is 1250 to 2499 Changes.

J.5
Half Peal

A Performance whose Length is 2500 to 4999 Changes.

J.6
Peal

A Performance whose Length is 5000 or more Changes.

J.7
Long Length

A Performance whose Length is 10000 or more Changes.

Further explanation: Long Lengths are also Peals -- they are a subset thereof.

J.8
Date Touch

A Performance whose Length corresponds to the year being commemorated.

Further explanation: The year being commemorated in a Date Touch is often the year in which it is rung. A Date Touch commemorating the year 2018 would have 2018 Changes and would also be a Quarter Peal.

J.9
Record Length

A Long Length that is the longest Length yet rung in a single Method or the same set of Methods, and that is also rung as described in Section M.

Further explanation: Record Lengths are also Peals -- they are a subset thereof.

J.10
Performance Report

A report of a Performance that includes the contents specified in Section P.

K
Method Extension
K.1
Parent Method

A Method at a given Stage from which it may be possible to derive related Methods at higher or lower Stages using extension or contraction techniques.

K.2
Extension

A Method at a higher Stage than a Parent Method that is related to the Parent Method by an extension technique.

K.2
Contraction

A Method at a lower Stage than a Parent Method that is related to the Parent Method by a contraction technique.
