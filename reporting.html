6. Performance Reporting

Z.
Definitions (to be re-ordered)
1.
Performance Title

A Performance Title takes the following form:
a) Number of Changes in the Performance;
b) 'Spliced' if the Performance was Spliced (see Section 6.A.3 below);
c) Name(s) of the Method(s) rung in the Performance. Where a Performance comprised more than one Method, this component of the Performance Title is optional -- the Method Names can be determined from the Performance Detail (see Section 6.Z.2 below);
d) Common Method Class(es), if any;
e) 'Variable Cover' if the Performance included Calls that affected Cover Bell(s);
f) Name(s) of the Stage(s) of the Method(s) rung in the Performance, listed in order of lowest Stage to highest Stage.
However, if the Effective Stage of the Performance is lower than the Stage of the highest Stage Method rung in the Performance, then only the Effective Stage is shown in the Performance Title.
This does not apply to Performances with Accepted Truth -- in this case, the two Stages rung are included in the Performance Title.

Examples:

  1. 1260 Plain Bob Minor
  2. 5040 Grandsire Triples
  3. 5040 Cambridge and Norwich Surprise Minor
  4. 5056 Spliced Plain and Canterbury Little Bob Major
  5. 5024 Spliced Surprise Major
  6. 5067 Spliced Differential Treble Place Caters
  7. 2520 Spliced Treble Dodging Royal
  8. 5070 Plain Doubles and Minor
  9. 5040 Minor
  10. 5016 Spliced Maximus
  11. 1320 Spliced Variable Cover Cinques and Maximus

Further explanation:

Where a Performance comprised a single Method, the Performance Title simplifies to:
a) Number of Changes in the Performance;
b) Method Title.
See examples 1 and 2 above.
Note that Grandsire does not have a Class word because it is one of the Methods with a Method Title exception (see Section 5.F).

Where a Performance comprised a small number of Methods / Variations, these may optionally be listed in the Performance Title. See examples 3 and 4 above. However this is not necessary since the same information is included in the Performance Detail (see Section 6.Z.2 below). Examples 5 to 11 above shown Performance Titles where Method / Variation Names are not included.

Where a Performance comprised Methods / Variations that all have the same Class Descriptor(s) (e.g. 'Surprise', 'Little Alliance', or 'Differential Little Bob') the Class Descriptor(s) are included in the Performance Title. See examples 5 and 6 above. In this case, the Class Descriptors do not need to be included in the Performance Detail (see Section 6.Z.2 below).

If the Methods / Variations do not all have the same Class Descriptors, but are all either Treble Dodging Methods, or Plain Methods / Variations, then 'Treble Dodging' or 'Plain' respectively is included in the Performance Title. See examples 7 and 8 above. In this case, the Class Descriptors are included in the Performance Detail (see Section 6.Z.2 below).

If the Methods / Variations do not all have the same Class Descriptors, and are not all either Treble Dodging or Plain Methods / Variations, then no Class words are included in the Performance Title. See examples 9 and 10 above. In this case, the Class Descriptors are included in the Performance Detail (see Section 6.Z.2 below).

Examples 8 and 11 above show Performance Titles when more than one Stage was rung.
Example 11 above shows a Performance Title when Variable Cover was rung.

2.
Performance Detail

The Performance Detail lists:
a) The number of Methods and, separately, the number of Variations rung in the Performance (as applicable);
b) The number of Changes rung of each Method and Variation;
c) The number of Changes of Method / Variation in the Performance;
d) Optionally, key features of the Composition such as 'atw' for 'all the work', and 'eld' for 'each lead different'.

Where a Performance comprised multiple Round Blocks (such as a Peal of 7 Round Extents of Minor), the Performance Detail may optionally, and instead of a) to c) above, list the Methods that were rung in each Round Block.

Examples:

  1. 8m: 640 each Bristol, Pudsey, Rutland, Superlative, Yorkshire; 608 each Cambridge, Lincolnshire, London; 120 com; atw.
  2. (1) Carlisle; (2) Beverley, Surfleet; (3) London, Wells; (4) York, Durham; (5) Norwich; (6) Ipswich; (7) Cambridge.

Further explanation:

The first example above is the Performance Detail for a Peal of Spliced Surprise Major.
8m indicates the Peal included 8 Methods.
120 com indicates there were 120 changes of Method.

If a Performance included both Methods and Variations, the Performance Detail would include, for example, 11m / 31v.
If the Performance just comprised Variations, the Performance Detail would include, for example, 21v.

The second example above is the alternative form of Performance Detail that may optionally be used for Performances comprising multiple Round Blocks.
This example is the Performance Detail for a Peal of Surprise Minor.
The numbers in brackets represent each Round Block in the Performance in the order they were rung, followed by the Methods (or Variations where applicable) that were rung in each Round Block.

A.
Content
1.

A Performance Report should include the following:
a) The Location where the Performance was rung;
b) The date on which the Performance was rung;
c) The Performance Title;
d) The Performance Detail;
e) Whether the Performance was on handbells or bells rung full-circle style;
f) The names of the ringers in the Performance and the number(s) of the bell(s) each rang; and
g) The names of any umpire(s) present.

Further explanation: The Location in a) above should be sufficient to identify unambiguously where the performance was rung, eg: St Stephen, Ambridge, Borsetshire; Casterbridge Town Hall, Wessex; 23 Railway Cuttings, East Cheam.

If a Performance spans midnight, the date in b) above should be the date on which the Performance ended.

In practice, the distinction in e) comes from separate reporting forms (e.g. on BellBoard) for handbells and tower bells.

2.

A Performance Report may also include additional information such as:
a) The society for which the Performance was rung;
b) Details of the Composition used in the Performance or a reference to it;
c) The name(s) of the composer(s);
d) The time the Performance took to ring;
e) The weight of the tenor for full-circle style ringing, or the size of the tenor for handbell ringing; and/or
f) Dedications and other footnotes.

Further explanation: Inclusion of the Composition per b) above is encouraged in all Performance Reports. Ideally the reader of a Performance Report should have enough information to be able to reproduce the Rows that were rung.

Time per d) above is encouraged for all Lengths, and should be included for Peal Lengths.

Tenor weight or size per e) above is encouraged for all Lengths, and should be included for Peal Lengths.

3.

A Performance involving more than one Method is described as Spliced if changes of Method occur at more than one distinct Row.

Further explanation: The format is [number of Changes] Spliced [common Class Descriptors] Stage(s), followed by the number of Methods rung, and the Name and number of Changes rung of each Method. E.g. ‘5024 Spliced Surprise Major (8m: 640 Bristol, 608 Cambridge, [etc])’.

If a multi-Method Performance is not Spliced, the format is [number of Changes] [common Class Descriptors] Stage(s), followed by the number of Methods rung, and the Name and number of Changes rung of each Method. E.g. ‘5040 Surprise Minor (7m: 1 extent each Cambridge, Beverley, Surfleet, [etc])’.

If the Methods that were spliced have more than one Class Descriptor, any common class elements are included in the Performance title, and the remainder are included in the Method list. E.g. 1280 Spliced Treble Dodging Major (2m: 672 Kent Treble Bob, 608 Cambridge Surprise), or 5008 Spliced Major (3m: 1360 Plain Bob, 2688 Yorkshire Surprise, 960 Duffield).

If Methods with more than one Stage were rung, each Stage is included in the Performance title. E.g. 5208 Spliced Triples and Major (2m: 2520 Stedman Triples, 2688 Bristol Surprise Major).

4.

The Performance Report of a Performance including one or more Methods that are not in the Methods Library should include the specification(s) and proposed name(s) of those Method(s).

5.

The report of a Performance that used simulated sound must state that it did.

B.
Performance Norms
1.

A Performance Report should state any aspect of the Performance that does not comply with the following norms, which will be assumed unless otherwise stated.

2.

The following are considered Norms for all reported Performances:
a) The Performance was a Round Block that started and ended in Rounds;
b) The Performance was a True Touch, or a Touch with Accepted Truth (as defined in Section 3.J);
c) The Performance was rung without interval;
d) On handbells the bells were retained in hand throughout the Performance;
e) The same person or persons rang each bell or bells continuously throughout the Performance;
f) Neither ringers nor conductor(s) used any physical aids to memory during the Performance;
g) No person not ringing provided any assistance in the execution of the ringing during the Performance, e.g. making calls, detecting or correcting errors;
h) Tower bells (or simulations thereof) were rung full-circle style;
i) Handbells (or simulations thereof) were rung in alternating up-strokes and down-strokes;
j) If Cover Bell(s) were used, these were in the highest Place(s) of the Rows;
k) Jump Changes were not used;
l) Only one Method was rung in any one Row;
m) The Effective Stage of the Performance was not lower than the highest Stage of Method rung in the Performance;
n) The Performance was consistent with this framework.

Further Explanation: For norm k), if Methods with Jump Changes were used, this will be evident from 6.A.1 d) above, since these Methods include "Jump" in their Titles. Therefore disclosure relating to Jump Changes is only required if Jump Calls (i.e. Calls involving Jump Changes) were used but Jump Methods (i.e. Methods involving Jump Changes) were not.

Norm m) does not apply to Performances with Accepted Truth.

3.

A performance with only one ringer should be witnessed by an umpire.

Further explanation: One person could ring a Performance of Minimus four in hand. The umpire provides corroboration that the Performance took place as reported.

Note that if this norm is not followed (i.e. no umpire was present for a single person performance), this will be evident from 6.A.1 g) and h) above, so no further disclosure is required.
