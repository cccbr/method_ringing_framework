Introduction

This document is a proposed replacement for sections D, E, F, G, I and J of the Central Council's Decisions.

Definitions

A Rows Show/hide detail & examples...
A.1 Row A sequence of bells in which each bell rings once and only once. Bells are incrementally numbered in order from highest to lowest pitch, and each is usually represented in a Row by a single digit or letter.

Example: 531246 is a 6 bell Row.

Bells are numbered in descending order of their pitch and may be referred to using either cardinal or ordinal numbers. Number 1 is used for the bell with the highest pitch (the treble), number 2 for the next highest pitch and so on. The bell with the lowest pitch is also referred to as the tenor. The numbering and naming of treble and tenor always relates solely to the set of bells comprising the Row, ignoring any others that may be rung at other times.

Rows are normally shown in columns so that the path of individual bells can be visualised by looking down the column. In order to keep the columns aligned, a fixed font can be used with single characters representing each bell. Conventionally '0' (zero) is used to represent 10 and either upper or lower case letters (excluding 'I', 'O' and 'X') are used for bells 11 to 33 as follows: 'E' (Eleven); 'T' (Twelve); 'A' (Thirteen); 'B' (Fourteen); 'C' (Fifteen); 'D' (Sixteen); 'F' (Seventeen); 'G' (Eighteen); 'H' (Nineteen); 'J' (Twenty); 'K' (Twenty-one); 'L' (Twenty-two); 'M' (Twenty-three); 'N' (Twenty-four); 'P' (Twenty-five); 'Q' (Twenty-six); 'R' (Twenty-seven); 'S' (Twenty-eight); 'U' (Twenty-nine); 'V' (Thirty); 'W' (Thirty-one); 'Y' (Thirty-two); 'Z' (Thirty-three)

A.2 Rounds A Row in which the bells ring in the order from the highest note to the lowest note.

Example: 12345678 is Rounds for an 8-bell Row.
A.3 Place The position of a bell within a Row.

Example: In the Row 142536, bell number 2 (which is also referred to as 'the 2nd') is ringing 3rd's Place.

More formally, a Place is a position where an individual bell can ring within a Row, referenced in the order of the first bell to ring to the last bell to ring, with one Place in a Row for every bell in that Row.

Positions are numbered using cardinal numbers, starting with 1 for the bell that rings first, and increasing sequentially for each next bell to ring e.g. the bell that rings second in a Row is in Position 2.

A Place is an equivalent term for a Position, but Places are numbered using ordinal numbers in the possessive form. E.g. the bell that rings second in a Row is in 2nd's Place. 2nd's Place here is a reference to the Place the 2nd bell is in when ringing Rounds.

However, when using Places, the bell in 1st Place is normally said to be Leading. And the bell in the last Place of the Row is said to be Lying. E.g. in the Row 142536, the Treble is Leading, and in the Row 2143658709, the 9th is Lying.

Note that in a Row containing one or more cover bells, the numbering of the Places may be modified to exclude the cover bell(s). E.g. when ringing Triples with a cover bell in 8th's Place, the bell in 7th's place may be said to be Lying.

Each bell in a Row rings in a different Position or Place to all other bells in that Row, and therefore there are the same number of Positions or Places in a Row as there are bells in that Row.

B Changes Show/hide detail & examples...
B.1 Change A transposition of bells from a first Row to a second Row, where both Rows are made up of the same set of bells.

A Change defines the transposition of all bells in a Row to their Places in the resulting Row, and a Change does not transpose two or more bells into the same Place in the resulting Row. E.g. a Change might transpose the bells from the Row 214365 to the Row 241356. Changes can be subdivided into types as follows:

B.1.1 Identity Change A Change that transposes each bell to the same Place in the second Row as it occupied in the first Row.

Example: The Change that transposes the bells from Row 54321 to Row 54321.
B.1.2 Adjacent Change A Change that transposes each bell either to the same Place in the second Row as it occupied in the first Row, or to a Place in the second Row that is immediately adjacent to the Place it occupied in the first Row, but which is not an Identity Change.

Example: The Change that transposes the bells from Row 21436587 to the Row 12346857.
B.1.3 Jump Change A Change that is neither an Identity Change nor an Adjacent Change.

Example: The Change that transposes the bells from Row 12345678 to Row 23415678 is a Jump Change because the Treble moves from 1st to 4ths Place which is not an adjacent Place.

B.2 Change Ringing The practice of ringing Rows where the progression between Rows is defined by Changes.

Call Changes (or 'Called Changes') is a form of Change Ringing, but not a form of Method Ringing.

C Methods Show/hide detail & examples...
C.1 Method A sequence of Changes, all operating on the same set of bells, that is used to produce a sequence of resulting Rows from an initial Row.

Example: -36-14-12-36-14-56,12 (Cambridge Surprise Minor).

Methods can be subdivided into Static Methods and Dynamic Methods, as follows:
C.1.1 Static Method A Method whose sequence of Changes is fixed and finite.

Example: -16-14-16-12 is an example of a Static Method on 6 bells. It can be represented by a fixed length place notation.

C.1.2 Dynamic Method A Method that is not a Static Method.

Example: Dixon's Bob Minor.

A Dynamic Method might specify that Rows are generated by successively applying the pair of Changes x16 except (a) if the Treble is Leading after an x Change, replace the next 16 Change with a 12 Change, and (b) if the 2nd or 4th is Leading after an x Change, replace the next 16 Change with a 14 Change. In this example, the sequence of Changes varies depending on the Rows generated so using a different starting Row could generate a different sequence of changes.

C.2 Method Name A name given to a Method that is unique for its Class and Stage.

A Method can be named when the requirements of Section N have been met.
C.3 Method Title The full description of a Method formed by concatenating its Method Name, Class Name and Stage Name.

Example: Plain Bob Doubles. 'Plain' is the Method Name, 'Bob' is the Class, and 'Doubles' is the Stage Name.
C.4 CRO Method Library A central collection, maintained by the CRO, of all named Methods recognized by the CRO
C.5 Rotation A Rotation is a Method whose Changes are the same as another Method, except that the Changes, when considered as a cycle, have been rotated to start at a different point in the cycle.

The Method Plain Bob Minimus has the Changes -14-14-14-12. The Method with the Changes -14-12-14-14 is a Rotation of Plain Bob Minimus.

C.6 Method Ringing The practice of ringing Rows without interval, where the progression between Rows is defined primarily by Methods, secondarily by Calls, with Rows optionally also including one or more Cover Bells.
D Cover Bells Show/hide detail & examples...
D.1 Cover Bell A bell that occupies a Place in a Row that is not one of the Places operated on by the Changes of the Method being rung.

Example: A Doubles Method is often rung with 6-bell Rows, where the Method's Changes operate on Places 1 to 5, and 6th's Place contains a Cover Bell (normally the tenor).

E Calls Show/hide detail & examples...
E.1 Call An instruction to replace consecutive Change(s) from a Method with different consecutive Change(s).

Example: The Change '12' when the treble is leading in Plain Bob Minor is replaced with the Change '14' when a bob has been called.

Most commonly a call replaces a single Change with another, but a call may also add Changes, remove Changes, or replace a number of Changes with a different number of Changes.

When a Call adds Changes, the additional Changes are added to the related Method's total number of Changes rung for Performance Reporting purposes.

When a Call affects the Changes of more than one Method (i.e. in spliced), the Composition should make clear how the Change count of each Method is affected so that the Performance can be accurately reported.

F Compositions
F.1 Composition An arrangement of Method(s) and Call(s), all operating on the same set of bells, that is used to produce a sequence of resulting Rows from an initial Row.
G Blocks Show/hide detail & examples...
G.1 Block A sequence of Rows, all composed of the same set of bells, that is produced from an initial Row using a Method or a Composition.

All Rows generated by a Block involve the same set of bells, and therefore all Rows have the same number of bells. An individual bell does not, for example, take part in some Rows, then stop for a period while other Rows are rung without it, and then join in again. When ringing a 6-bell Block at a tower with, for example, 10 bells, the same 6 bells ring in all Rows of the Block, such that the other 4 bells in the tower do not ring at any point in the Block.

There is no requirement to use whole multiples of a Method's Changes - any portion of a Method's Changes can be used in the generation of a Block.

While Rows (and therefore Blocks) are typically generated using one Method at a time, it is possible for a single Row to be generated using two or more Methods. For example a 12-bell Row could be generated by using a Triples Method that operates on Places 1 to 7, and a Doubles Method that operates on Places 8 to 12.

There are no Rows in a Block that consist solely of Cover Bells. At least one Method is used in the generation of every Row of a Block, and a Method is still active even if its Places have been replaced or extended by a Call.

G.2 Round Block A Block whose final Row is the same as its initial Row.
G.3 Plain Lead A Block produced, starting from an initial Row, by applying a Method's sequence of Changes once, without using any Calls.

The place notation for Stedman Doubles is 3.1.5.3.1.3.1.3.5.1.3.1. When these changes are applied once to a starting row, the result, by definition, is one Plain Lead. Note that if a method has palindromic symmetry, it may be recorded in shorthand in a Method Library in a form such as &x16x14+12. The '&' indicates that the portion before the '+' must also be reversed (except for the last change), before then adding the portion after the '+'. In full this method's place notation is therefore x16x14x16x12 and this is the place notation that defines one plain lead.

G.4 Plain Course A Block produced, starting from an initial Row, by applying a Method's sequence of Changes repeatedly, without using any Calls, until a Round Block is obtained.

Example: The place notation for Bastow Little Bob Minimus is x12x14. This place notation produces one Plain Lead, bringing up the row 1423 as shown in the diagram (the first underlined row after the underlined Rounds at the top). If this place notation is applied twice more, Rounds is produced.

Bastow Little Bob Minimus therefore has 3 Plain Leads in its Plain Course.

G.5 Length The Length of a Block is the number of Changes that were used to produce the Block.

Length is measured in number of changes instead of number of rows to avoid the question of whether the opening or closing rounds, or neither or both, are included in the count.

In this example block, it can be seen there are a total of 9 rows, including both the initial rounds and the final rounds. However, 8 changes (as represented by place notations on the left) are used to produce this block. The block therefore has a length of 8 changes.

G.6 Fixed Cover Place A Place in a Block in which the same Cover Bell rings in all Rows of that Block.

Example: Assume Plain Bob Doubles is being rung on the front 5 of a ring of 8, and Cover Bells are ringing in 6th's, 7th's and 8th's Places. Calls are used in the Composition such that (a) occasionally the bells in 5th's and 6th's Places are called to swap, and (b) occasionally, at other times to (a), the bells in 6th's and 7th's Places are called to swap.

In this ringing, the bell ringing in 8th's Place is always the same bell because it is not ringing a Method and it is never affected by a Call. 8th's Place is therefore a Fixed Cover Place.

G.7 Variable Cover Place A Place in a Block in which a Cover Bell rings for at least one Row of that Block, but which is not a Fixed Cover Place.

In the example for G.6 above, 6th's and 7th's Places are Variable Cover Places because they contain Cover Bells for at least one of the Rows (and in fact for this example, for all of the Rows), but they are not Fixed Cover Places because the same Cover Bell is not ringing in each of 6th's and 7th's Places all the time.

Places 1 - 5 in the example for G.6 above never contain a Cover Bell, so they are neither Fixed Cover Places nor Variable Cover Places.

G.8 Extent The complete set of distinct Rows possible for a given set of bells.

The number of Rows in an Extent for a given set of bells is the factorial of the number of bells in the set. E.g. for 4 bells, the Extent has 4! Rows = 1 * 2 * 3 * 4 = 24 Rows.

Note that while a Block may contain one or more Extents, this definition of Extent neither implies that the Rows are in any particular order, nor that the Rows making up an Extent are necessarily contiguous within a Block.

H Stages Show/hide detail & examples...
H.1 Stage The number of bells.

A term that applies to Rows, Changes, Methods, Compositions and Blocks:

The Stage of a Row is the number of bells in that Row.

The Stage of a Change is the number of bells on which that Change operates.

Since a Method comprises Changes that all operate on the same set of bells, a Method has the Stage of its underlying Changes.

Since a Composition is used to produce Rows that comprise the same set of bells, a Composition has the Stage of the Rows it produces.

Since a Block comprises Rows that are all composed of the same set of bells, a Block has the Stage of its underlying Rows.

H.2 Stage Name

Names that are given to Stages:

3 = Singles; 5 = Doubles; 7 = Triples; 9 = Caters; 11 = Cinques; 13 = Sextuples; 15 = Septuples; etc.

2 = Two; 4 = Minimus; 6 = Minor; 8 = Major; 10 = Royal; 12 = Maximus; 14 = Fourteen; 16 = Sixteen; etc.
H.3 Effective Stage

The number of Places remaining in a Block after any Fixed Cover Places have been excluded.

The Effective Stage of a Composition is the same as the Effective Stage of the Block it produces.

In the example for G.6 above, the Effective Stage would be 7 because only 8th's Place is a Fixed Cover Place.

I Truth Show/hide detail & examples...
I.1 True A Block is True if each of the possible Rows for the Block's Effective Stage occurs either n or n+1 times, for a given value of n greater than or equal to zero.

When the number of Rows in a Block is less than or equal to the number of Rows in the Extent of the Block's Effective Stage, the definition of True simplifies to 'a Block that has no repeated Rows'. However, the definition of True becomes more involved when a Block has more Rows than the number of Rows in the Extent of the Block's Effective Stage. Following are examples of three different scenarios: 

Example 1:

In the Block in Example 1, a Minor Method is being rung on the front 6 with a Cover Bell in 7th's Place. The bell in 7th's Place (the 7th) is never affected by a Call. 7th's place is therefore a Fixed Cover Place and is excluded when determining truth: 

    123456
    ------
    213546
    231564
    321654
    312645
    132465
    123456
    -------

There are 6 remaining Places, so the Effective Stage of the Block is 6 (i.e. Minor).

The Extent of a Block with an Effective Stage of 6 has 6! Rows = 720 Rows.

6 of the Rows in the Extent of the Block's Effective Stage appear in the above Block once. 714 of the Rows in the Extent of the Block's Effective Stage appear in the above Block zero times.

Since Rows in the Extent of the Block's Effective Stage appear in the Block either once or zero times, the requirement that each of the possible Rows for the Block's Effective Stage occurs either n or n+1 times, for a given value of n greater than or equal to zero, is met, and so the Block is True.

Since all Rows of the Effective Extent appear either once or twice, the 'either n or n+1 times' requirement is met and the Block is True.

In the Block in Example 3, a Singles Method is rung on the front 3 for the first 6 Changes, together with a Cover Bell in 4th's Place. Then there is a switch to a Minimus Method for the remaining 8 Changes of the Block.

There are no Fixed Cover Places in this Block. 4th's Place is a Variable Cover Place because Methods at both Stages 3 and 4 are used. So there are no Places to exclude when determining truth.

The Effective Stage of the Block is therefore 4.

The Extent of a Block with an Effective Stage of 4 has 4! Rows = 24 Rows.

2 of these 24 Rows appear twice, 10 of these 24 Rows appear once, and 12 of these 24 Rows appear zero times.

Since all Rows of the Effective Extent appear zero times, once or twice in the Block, the 'either n or n+1 times' requirement is not met (since some rows appear n+2 times) and the Block is False.

J Performances Show/hide detail & examples...
J.1 Performance The successful ringing of a Block where the band strives to maintain a high standard of ringing, and errors in ringing or calling are corrected quickly.

Performances of 1250 Changes or greater are often published to the Ringing Community.

More fully, this term would be 'Method Ringing Performance'. Performances are often distinguished from practice ringing by the intent for the ringing to be heard by members of the public, and/or for the ringing to be reported to the ringing community if the ringing was of Quarter Peal Length of greater. 'Method Ringing Practice', by contrast, is often for the purpose of teaching and developing ringers.

J.2 Touch A Performance of any Length.

Performances of 1250 Changes or greater are often published to the Ringing Community.
J.3 Short Touch A Performance whose Length is 1 to 1249 Changes.
J.4 Quarter Peal A Performance whose Length is 1250 to 2499 Changes.
J.5 Half Peal A Performance whose Length is 2500 to 4999 Changes.
J.6 Peal A Performance whose Length is 5000 or more Changes.
J.7 Long Length A Performance whose Length is 10000 or more Changes.

Long Lengths are also Peals - they are a subset thereof.
J.8 Date Touch A Performance whose Length corresponds to the year being commemorated.

The year being commemorated in a Date Touch is often the year in which it is rung. A Date Touch commemorating the year 2018 would have 2018 Changes and would also be a Quarter Peal.

J.9 Record Length A Long Length that is the longest Length yet rung in a single Method or the same set of Methods, and that is also rung as described in Section M.

Record Lengths are also Peals - they are a subset thereof.
J.10 Performance Report A report of a Performance that includes the contents specified in Section P.
K Method Extension and Contraction
K.1 Parent Method An existing Method at a given Stage for which related Methods at other Stages exist or are being sought.
K.2 Extension A Method at a higher Stage than a Parent Method that is related to the Parent Method by an extension technique.
K.3 Contraction A Method at a lower Stage than a Parent Method that is related to the Parent Method by a contraction technique.
