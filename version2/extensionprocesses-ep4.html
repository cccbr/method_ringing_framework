Appendix D2. Method Extension Process EP4 (DRAFT ONLY)

4.
Extension Process 4 (EP4)

EP4 is similar to EP3 except that it gives greater flexibility in selecting which set of changes in a lead are copied and possibly extended to generate the extension. In EP3, the set of copied changes is limited to specified sections, where sections are based on the position of the hunt bell. EP4 allows the copied section to begin at any change in the lead.

EP4 is designed to extend Parent Methods that are Hunters with a Hunt Bell that has a non-Little Path. The resulting Extension is an even number of Stages higher than the Parent Method.

Prerequisites

The prerequisites for this Extension Process are as follows:

  1. The Parent Method is a non-Little Plain or Treble Dodging Method, or a non-Little Alliance Method with a Hunt Bell Path formed from these two. This Extension Process may also be applied to some other forms of Hunter with a Hunt Bell that has a non-Little Path.
  2. The Parent Method does not use Jump Changes.

Steps

The steps to implement this Extension Process are shown using the following example where Cambridge Surprise Minor is the Parent Method. Further instructions on these steps are provided after the example.

Further instructions

  1. Rotate the place notation of the method into a form that includes both pivot points for palindromic methods, and starts at the lowest pivot place of the hunt bell. For pallindromic methods, this typically means starting with the lead end change and finishing with the half lead change. Application of EP4 to non-pallindromic methods is covered further below.
  2. The place notation is divided into three components: places made by the hunt bell (line 2 in the example); places made below the hunt bell (line 7); and places made above the hunt bell (line 12). The places above the hunt bell are reversed and inverted so that they can be treated identically to the hunt bell places and the places below the hunt bell.
  3. For each of the three components, 4 parameters are selected:
    • Overlap start: This determines the starting change of the section of the method's place notation that will be copied and possibly extended in the extension.
    • Overlap length: This determines how many changes will be added to a half-lead of the method.
    • Mode: This is defined in Appendix D.A.3.
    • Place increment: This specifies how many places are added to the corresponding place in the parent method. For example, if the place increment is 2 and the parent method contains the place 5, then this place may become 7 (5 + 2) in the extension.
  4. The combination of overlap length and place increment determines the stage increment of the extension. If the method is Plain, the overlap length is 2, and the place increment is 2, then the extension will be 2 stages higher than the parent method (e.g. Minor extends to Major). If the method is Treble Dodging, the overlap length is 4, and the place increment is 2, then the extension will also be 2 stages higher than the parent method. Extension in stage increments greater than 2 can be achieved with different overlap length and place increment parameters. For example, to extend a treble dodging method in steps of 4 stages, use an overlap length of 8 and a place increment of 4.
  5. Each of the 3 components above are then split into two overlapping segments based on the overlap start and overlap length parameters. The places in the first segment are left unchanged, and the places in the second segment are extended according to the mode and the place increment. Places in the second segment that are less than or equal to the mode are left unchanged. All other places in the second segment are increased by the place increment. For each component, the two segments are then recombined to form the hunt places, below places and above places of the extension.
  6. After reversing and inverting the extended places above the hunt bell back to their original orientation, the 3 components are then recombined into a single place notation for the extension.
  7. Apply the same process above to the extension to obtains further extensions at higher stages.

Requirements

An Extension produced using EP4 is only valid if:

  1. The Extension has the same Symmetry (see Section 4.B) as the Parent Method.
  2. The Extension has the same number of Hunt Bells as the Parent Method, unless the Parent Method only has Hunt Bells and has no Working Bells, in which case the Extension also only has Hunt Bells.
  3. The Extension has the same number of Cycles of Working Bells as the Parent Method.
  4. If the Parent Method comprises two or more Cycles of Working Bells of equal size, the Extension has this same feature.
  5. If the Parent Method has Plain Bob Leadheads, the Extension also has Plain Bob Leadheads.
    If the Parent Method does not have Plain Bob Leadheads, the Extension also does not have Plain Bob Leadheads.
  6. The Extension Construction used, when applied to the Parent Method in question, creates an Extension Path containing a minimum of 3 Methods (including the Parent Method) with Stages that are less than or equal to Stage 24.

Extension Construction

  1. The Extension Construction for referring to this Extension Process is:
    EP4 -
    [Overlap start for hunt places] : [Overlap length for hunt places] : [Mode for hunt places] : [Place increment for hunt places] /
    [Overlap start for below places] : [Overlap length for below places] : [Mode for below places] : [Place increment for below places] /
    [Overlap start for above places] : [Overlap length for above places] : [Mode for above places] : [Place increment for above places]
  2. The Extension Construction used in the example above is therefore EP4-1:4:1:2/5:4:1:2/7:4:1:2

Other considerations

  1. The example above uses an overlap length of 4 and a place increment of 2, giving an extension that is 2 stages higher that the parent since the hunt bell follows a treble dodging path. If an overlap length of 8 and a place increment of 4 were used, the extension would be 4 stages higher than the parent. Superlative Surprise Major is an example of a Method that extends in steps of 4 stages. The Extension Construction for Superlative's Extension Process is EP4-1:8:1:4/9:8:3:4/9:8:3:4
  2. Different overlap lengths and place increments may be used for the different components. For example, an overlap length of 8 and a place increment of 4 could be selected for the hunt places and the below places, and an overlap length of 4 and a place increment of 2 could be selected for the above places. In this case, the extension process for the above places is applied twice for every one time the process is applied to the hunt places and below places, and the resulting extensions would have a stage increment of 4. For example, there is an Extension Process for Cambridge Surprise Major that uses steps of 2 above the treble and steps of 4 below the treble. The Extension Construction for this Extension Process is EP4-1:8:1:4/4:8:4:4/13:4:1:2
  3. When the same results are obtained using different overlaps starts, designate the lowest start as the one that was used. This ensures that whatever Stage Method is taken as the Parent Method in a set of Methods that are on the same Extension Path, the same Extension Construction will result.
  4. When two or more different Modes give the same results, designate the lowest mode as the one that was used.

Non-pallindromic methods

  1. The example and descriptions above assume the Parent Method has Palindomic Symmetry about the Leadend Change and the Halflead Change. For non-Palindromic Methods, EP4 may be applied separately in both halves of the Plain Lead, subject to the following:
  2. If the Places Made below the treble are Palindromic, or the Places Made above the treble are Palindromic, then an Extension Construction is selected that retains that symmetry in the Extension.
  3. If the Places Made above the treble in the first half of the Plain Lead have Rotational Symmetry with the Places Made below the treble in the second half of the Plain Lead, or the Places Made below the treble in the first half of the Plain Lead have Rotational Symmetry with the Places Made above the treble in the second half of the Plain Lead, then an Extension Construction is selected that retains that symmetry in the Extension.
  4. An example of an Extension Construction for a non-Palindromic Method is EP4-1:4:1:2/5:4:1:2/7:4:1:2/13:4:1:2/17:4:1:2/19:4:1:2 The sections are respectively:
    • Parameters for the hunt places for the first half lead
    • Parameters for the below places for the first half lead
    • Parameters for the above places for the first half lead
    • Parameters for the hunt places for the second half lead
    • Parameters for the below places for the second half lead
    • Parameters for the above places for the second half lead
