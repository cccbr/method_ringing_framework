B.
General
1.

EP4 splits the place notation of a parent method into two overlapping segments that, after certain modifications, are then rejoined to form the place notation of the extension.

For hunters, the places made above and below the hunt bell, and by the hunt bell itself, are treated separately and identically.

EP4 replaces both EP3 and EP1.

EP4 can operate in both single mode and tri mode as described below.

EP4 supports the contraction of parent methods to a lower stage. Note that it is not a requirement for a contraction to be able to be re-extended back to the parent method.

2.

EP4 applies to the following types of methods:

(1) Non-Little Hunters. If there is more than one hunt bell, they all course (meaning there is no room for a working bell to make a place between them). The hunt bell(s) only make places in first's place and the last place of the method's stage. When EP4 is applied to these methods, the lead length of the extension will be longer than the parent.

(2) Principles (i.e. methods with no hunt bells). When EP4 is applied to these methods, the lead length of the extension can either be the same as the parent, or it can be longer than the parent, depending on how EP4 is applied (details below).

(3) Little methods. If there is more than one hunt bell, they all have a little path and they all course (meaning there is no room for a working bell to make a place between them). The hunt bell(s) only make places in the lowest place of their path and in the highest place of their path. When EP4 is applied to these methods, the lead length of the extension can either be the same as the parent, or it can be longer than the parent, depending on how EP4 is applied (details below).

EP4 supports the contraction of parent methods to a lower stage. Note that it is not a requirement for a contraction to be able to be re-extended back to the parent method.
